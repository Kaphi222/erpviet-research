Cấu hình toàn site

📊 Kết nối với Google Sheets qua Apps Script. Tải dữ liệu mới nhất, export JSON backup, hoặc restore Sheet từ file JSON cũ.

☁️ Tải từ Google Sheets

Lấy dữ liệu mới nhất từ Sheet và xem trước trong bảng bên dưới.

⬇️ Export JSON (Backup)

Tải dữ liệu từ Sheet về dạng JSON để commit lên GitHub. Đây là bản backup version-controlled.

↑ Restore từ JSON → Google Sheets

Upload file JSON cũ để ghi đè nội dung trong Sheet (dùng để rollback về version cũ). Cẩn thận: sẽ xóa toàn bộ dữ liệu Sheet hiện tại.

⚙️ Cấu hình Apps Script (cần làm 1 lần)

Cập nhật Apps Script hiện có để hỗ trợ CMS (đọc tab theo tên + restore). Mở Apps Script → thay toàn bộ bằng code bên dưới → Deploy lại (New deployment).

Xem code Apps Script cần cập nhật → 
const SHEET_ID = 'YOUR_GOOGLE_SHEET_ID'; // thay bằng ID Sheet của bạn

function doGet(e) {
  const sheetName = e.parameter.sheet;
  const ss = SpreadsheetApp.openById(SHEET_ID);

  // CMS read: ?sheet=Nganh or ?sheet=CaseStudies
  if (sheetName) {
    const sheet = ss.getSheetByName(sheetName);
    if (!sheet) return json({error:'Sheet not found'});
    const rows = sheet.getDataRange().getValues();
    const headers = rows[0];
    const data = rows.slice(1)
      .filter(r => r[0] !== '')
      .map(r => {
        const obj = {};
        headers.forEach((h, i) => { if (h) obj[h] = r[i]; });
        return obj;
      });
    return json(data);
  }

  // Booking form write: ?name=...&email=...
  const p = e.parameter;
  if (p.name) {
    const bookingSheet = ss.getSheetByName('Responses') || ss.getSheets()[0];
    bookingSheet.appendRow([
      new Date(), p.name, p.company, p.email, p.phone,
      p.role, p.industry, p.topics, p.date, p.time,
      p.current, p.size, p.note
    ]);
  }
  return json({status:'ok'});
}

function doPost(e) {
  const p = JSON.parse(e.postData.contents);
  const ss = SpreadsheetApp.openById(SHEET_ID);
  const sheet = ss.getSheetByName(p.sheet);
  if (!sheet) return json({error:'Sheet not found'});
  const rows = p.data;
  sheet.clearContents();
  const headers = Object.keys(rows[0]);
  sheet.appendRow(headers);
  rows.forEach(r => sheet.appendRow(headers.map(h => r[h] !== undefined ? r[h] : '')));
  return json({status:'ok'});
}

function json(data) {
  return ContentService
    .createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}
Tabs cần có trong Sheet: Responses (booking), Nganh, CaseStudies. Headers phải khớp với field trong JSON.

📌 Workflow: Chỉnh sửa trên Admin Panel → Xuất JSON → Commit lên GitHub → GitHub Pages tự cập nhật

📋 Workflow cập nhật nội dung

  1. Chỉnh sửa dữ liệu trong Admin Panel
  2. Nhấn Xuất JSON để tải file về máy
  3. Thay file cũ trong thư mục /data/
  4. Commit và push lên GitHub
  5. GitHub Pages tự động cập nhật (~1-2 phút)
git add data/industries.json
git commit -m "Update: thêm ngành XYZ"
git push origin main

🚀 Publish lên GitHub Pages

  1. Tạo repository mới trên GitHub
  2. Push toàn bộ code lên branch main
  3. Vào Settings → Pages
  4. Source: Deploy from branch
  5. Branch: main, Folder: / (root)
  6. Save → chờ ~2 phút để deploy
Site URL: https://<username>.github.io/<repo-name>

🌐 Cấu hình tên miền riêng

  1. Mua tên miền (Inet, Matbao, Namecheap...)
  2. Thêm CNAME vào file /CNAME trong repo
  3. Vào DNS manager, thêm CNAME record:
    www → <username>.github.io
  4. Vào GitHub Settings → Pages → Custom domain
  5. Bật "Enforce HTTPS" khi DNS đã propagate
CNAME file: www.erpvietresearch.com

✏️ Thêm case study mới

  1. Nhấn + Thêm case trong tab Case Studies
  2. Điền đầy đủ thông tin
  3. Đặt Featured = true nếu muốn hiện spotlight
  4. Xuất JSON → thay file /data/case-studies.json
  5. (Tùy chọn) Tạo thêm trang chi tiết trong /pages/case-studies/
Mỗi case cần: id, slug, company, industry,
title, summary, results[], date, featured
